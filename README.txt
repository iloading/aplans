Projeto final conjunto das UC's de LAB4 , CI & II. A aplans é uma app desenhada e pensada pelo nosso grupo com o objetivo de auxiliar as pessoas do bairro de Santiago a organizar eventos para que a comunidade se aproxime e haja um melhor ambiente entre moradores. Foi usado, como base o projeto de LAB2 pois achámos que o trabalho já desenvolvido seria útil, pelo que se houver ficheiros que pareçam não pertencer ao aplans, provavelmente são ficheiros que restaram do "template" de LAB2.
